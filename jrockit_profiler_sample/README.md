How to transform XQuery to XML in Java
========================
Oracle JRockit contains a great API called JMAPI – Monitoring and Management APIs. With JMAPI you can get the Profiling System (low overhead profiler) and you can to perform lightweight profiling of the JVM.
This profiling system will tell you how many times the method has been used, and total time spent in this method.

### Setup
* The source code is eclipse project
* JRockit 1.6 (R28)
